Linux Learning Guide

1. Finding Files using "locate"

Example: For finding any file, you can use locate specific word.

Bash
locate filename
Output
/mnt/c/rtools45/usr/share/man/man3/curl_mime_filename.3.gz

Example: For finding specific file formats, use locate .fileformat.

Bash
locate .txt
locate .pdf
Output
/usr/share/vim/vim81/tutor/README.txt
/usr/share/doc/shared-mime-info/shared-mime-info-spec.pdf
Note

Always use sudo updatedb if you cannot find newly created files.
